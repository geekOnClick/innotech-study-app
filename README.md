# Трекер задач

Учебное приложение на React + TypeScript. Данные приходят через моковый API без реального бекенда: методы возвращают `Promise` с небольшой задержкой, поэтому интерфейс ведет себя как при обычных сетевых запросах.

## Запуск

```bash
npm install
npm run dev
```

Если на Linux появляется ошибка `ENOSPC: System limit for number of file watchers reached`, запустите dev-сервер через polling:

```bash
npm run dev:poll
```

## Что есть в приложении

- загрузка стартовых задач через моковый запрос;
- создание, редактирование, удаление и переключение готовности задачи;
- фильтрация по статусу и приоритету;
- поиск по названию, описанию и исполнителю;
- счетчики задач и состояния загрузки/сохранения.

Моковый API находится в `src/mockApi.ts`.
